# Quickstart: Sistema de Dificuldade para Minecraft

## Como executar

1. Abra o arquivo `src/index.html` em qualquer navegador web

## Novas funcionalidades

### Seleção de Dificuldade
- Três botões: Fácil, Médio, Difícil
- Clique em um para selecionar

### Geração de Desafio
- Selecione a dificuldade
- Clique em "Gerar Desafio"
- Receba um desafio dinâmico baseado na dificuldade

### Geração Dinâmica
- Combina objetivos + restrições + condições
- Maior variety de desafios
