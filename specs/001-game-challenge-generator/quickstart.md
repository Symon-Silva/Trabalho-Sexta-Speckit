# Quickstart: Game Challenge Generator

## Como executar

1. Abra o arquivo `src/index.html` em qualquer navegador web

## Estrutura dos arquivos

```
src/
├── index.html    # Página principal
├── styles/
│   └── style.css  # Estilos
└── scripts/
    └── app.js    # Lógica
```

## Uso

1. Abra a página no navegador
2. Selecione um jogo da lista
3. Clique no botão "Gerar Desafio"
4. Um desafio aleatório será exibido

## Para desenvolvimento

Edite os arquivos em `src/` conforme necessário. Não requer build ou servidor.
